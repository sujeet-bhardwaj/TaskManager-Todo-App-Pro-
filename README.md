# 📝 Todo App Pro

A modern and responsive Todo Application built with **React.js** that helps users manage daily tasks efficiently. The application allows users to create, edit, delete, and organize todos with persistent storage using the browser's **Local Storage**.

---

## 🚀 Live Demo
🔗 **Live Website:** [Todo App Pro](https://sujeettodo.netlify.app/)
---




# ✨ Features

* ✅ Add new tasks
* ✏️ Edit existing tasks
* 🗑️ Delete tasks
* ✔️ Mark tasks as completed
* 📋 View all tasks
* 💾 Data persists using Local Storage
* 📱 Fully Responsive Design
* ⚡ Fast and Lightweight
* 🎨 Clean User Interface



# 🛠️ Built With

* React.js
* HTML5
* CSS3
* JavaScript (ES6+)
* Local Storage API

# 💻 Installation

Clone the repository

```bash
git clone https://github.com/sujeetbhardwaj/TaskManager-Todo-App-Pro.git
```

Navigate to the project

```bash
cd YOUR_REPOSITORY_NAME
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

---

# 💾 Local Storage

This project uses the browser's **Local Storage API** to save todo data.

Benefits:

* Data remains after refreshing the page
* No backend required
* Fast data access
* Simple and lightweight persistence
